## Installation ##

Add the following to your composer file:

```json
   "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sinevia/php-library-kvs.git"
        }
    ],
    "require": {
        "sinevia/php-library-kvs": "dev-master"
    },
```
